# UKSRC Standards and Procedures

This repository contains the technical policies, processes, and guidelines used across the UKSRC. Its purpose is to provide a single, version-controlled source of truth for engineering, CI/CD, security, and operational standards.

### Contents

- CI/CD policies

- Engineering and coding standards

- Operational procedures

- Governance and review processes

- Documentation templates

### Contributing

- All changes must be made via pull request.

- Follow the established review process.

- Ensure documents remain clear, versioned, and auditable.
